## Primary Navigation By Role

### Admin / Manager

Show:

1. `Home`
2. `Dough Prep`
3. `Admin Panel`

### PizzaMaker

Show:

1. `Home`
2. `Dough Prep`

Do not show:

- `Admin Panel`
- user management
- weekly closing
- losses / analytics
- advanced data maintenance

## Landing After Login

- `Manager` and `Admin` land on `Home`
- `PizzaMaker` may land on `Home` or a simplified `Dough Prep`, but must still see only the reduced navigation

## Admin Panel Contents

Admin Panel is where advanced flows live:

- `Users`
- `Weekly Closing`
- `Losses`
- `Dough Quality`
- `Prep Dough Data`
- `Recommendations`
- future history/settings screens

## Dough Prep Role Behavior

### Manager / Admin

- can calculate guidance
- can save recommendation
- can create dough tasks
- can open advanced quality flows
- can close weeks

### PizzaMaker

- can see simplified operational cards
- can complete allowed dough tasks
- can work ball dough
- can work reball if current permissions allow
- cannot access admin-only control surfaces

## UX Principle

Role differences should change what is visible, not make the product feel like a different system.
